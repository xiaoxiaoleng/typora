##### 包管理工具

```python
#列出可用的 Python 版本：
pyenv versions
#全局设置默认 Python 版本
pyenv global <version>
#设置项目级别的 Python 版本：
pyenv local <version>
#使用安装所需的版本
pyenv install <version>
#安装列表
pyenv install --list
#卸载指定版本
pyenv uninstall 3.7.9
```

##### 安装依赖

```python
pip install -r requirements.txt
```
